# Z-Tally

<p align="center">
  <img src="./release/icon-240.png" alt="Z-Tally app icon" width="160">
</p>

An offline tally counter app for Amazfit watches running Zepp OS. Z-Tally provides three independent counters, local history, and a compact interface designed for round watch displays.

Z-Tally 是一款面向 Zepp OS Amazfit 手表的离线按压计数器应用，提供三组独立计数、本地历史记录，以及适配圆形表盘的简洁操作界面。

- Works fully offline / 完全离线运行
- No account, ads, or cloud sync / 无账号、广告或云同步
- English and Simplified Chinese / 支持英文与简体中文
