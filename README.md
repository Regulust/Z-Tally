# Z-Tally

<p align="center">
  <img src="./release/icon-240.png" alt="Z-Tally app icon" width="160">
</p>

An offline tally counter app for Amazfit watches running Zepp OS. Z-Tally provides three independent counters, local history, a SecondaryWidget, and layouts for round and 390×450 square displays.

Z-Tally 是一款面向 Zepp OS Amazfit 手表的离线按压计数器应用，提供三组独立计数、本地历史记录、副屏计数，以及适配圆屏和 390×450 方屏的操作界面。

- Works fully offline / 完全离线运行
- No account, ads, or cloud sync / 无账号、广告或云同步
- 8 languages: English, Simplified/Traditional Chinese, Spanish, Portuguese, Italian, German, and Russian / 支持 8 种语言
- Round and 390×450 square displays / 支持圆屏与 390×450 方屏
- SecondaryWidget quick counting / 支持副屏快速计数
